# Start Session

Run:

```bash
python scripts/session/cli.py start $ARGUMENTS
```

Display the full briefing output to the user.

After displaying the briefing, begin implementing the work item according to the spec file shown. Follow the patterns in ARCHITECTURE.md and the guidelines in CLAUDE.md.
